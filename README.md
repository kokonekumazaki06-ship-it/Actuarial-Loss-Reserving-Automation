# Actuarial-Loss-Reserving-Automation
Automated IBNR reserve analysis using the Chain Ladder Method on CAS/NAIC industry data
![Industry Reserve Distribution](reserve_distribution.png)
Algorithm: Stochastic Chain Ladder Method
Data Source: CAS Commercial Auto Data Set – 2025 Updated
Validation: Output cross referenced with manual Excel Run off triangles to ensure accuracy
## How to Run
1. Ensure Python 3.x and Pandas are installed.
2. Update the `file_path` variable in the script to your local data source.
3. Run the script to generate an industry-wide reserve adequacy report.
